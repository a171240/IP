## 目的
帮助实体店老板建立完善的员工培训体系，包括：
1. 新员工入职培训体系
2. 在岗技能提升方案
3. 管理人才培养计划
4. 企业文化传承机制
5. 培训效果评估体系
最终目标是打造学习型组织，提升员工能力和忠诚度，降低人员流失率。

## 思考框架
在设计培训方案前，请分析：

1. **问题的核心意图和边界条件**
   - 员工现状分析（学历、年龄、经验）
   - 培训需求评估（技能差距、发展需求）
   - 可用资源评估（时间、预算、场地）
   - 业务特点和要求

2. **可能的解决方案及其优缺点**
   - 集中培训 vs 分散培训的效果
   - 内部培训 vs 外部培训的选择
   - 线上学习 vs 线下面授的结合
   - 理论讲解 vs 实操演练的比例

3. **需要考虑的边缘情况**
   - 员工学习意愿不高
   - 培训与工作时间冲突
   - 培训后人才流失
   - 不同岗位差异化需求

### 📚 新员工7天培训计划

**Day 1：欢迎加入**
```
上午（3小时）：
- 破冰游戏：奶茶名字接龙
- 企业文化：品牌故事分享
- 门店参观：熟悉环境
- 团队介绍：一对一认识

下午（3小时）：
- 基础知识：奶茶行业概况
- 产品体验：品尝全系列产品
- 作业：写下最喜欢的3款产品及理由

培训方式：讲解+体验+分享
```

**Day 2：产品认知**
```
上午：原料知识
- 茶叶基础：红茶/绿茶/乌龙茶
- 配料介绍：珍珠/芋圆/布丁制作
- 互动：蒙眼品茶游戏

下午：调配基础
- 基础手法：摇茶、封口、装杯
- 实操练习：制作标准奶茶
- 考核：3分钟内完成一杯

工具：原料卡片、操作视频、练习材料
```

**Day 3：服务流程**
```
全天实训：
□ 08:00 开店准备SOP
□ 09:00 点单系统操作
□ 10:00 标准服务7步骤
□ 14:00 角色扮演练习
□ 16:00 收银对账流程
□ 17:00 闭店清洁标准

亮点：老员工示范+新人实操+及时纠正
```

**Day 4：销售技巧**
```
专题培训：
1. 黄金销售话术
   "今天想喝点什么？"
   "我们的招牌是..."
   "要不要加点小料？"

2. 连带销售技巧
   - 观察顾客特征
   - 推荐搭配产品
   - 优惠活动介绍

3. 实战演练
   - 模拟20个场景
   - 录像回放点评
```

**Day 5：问题处理**
```
案例教学：
- 顾客说太甜了怎么办？
- 等待时间长如何安抚？
- 撒了饮料如何处理？
- 遇到挑剔顾客怎么办？

处理原则：
1. 先道歉，后解决
2. 换位思考，理解顾客
3. 超预期补偿
4. 及时上报
```

**Day 6：综合实习**
```
全天跟岗：
- 早班：跟随店长学习
- 午高峰：独立接单（督导）
- 下午：复盘问题，补充培训
- 晚高峰：考核实战

过关标准：
✓ 独立完成30单
✓ 无重大失误
✓ 顾客满意度100%
```

**Day 7：考核转正**
```
理论考试（40%）：
- 企业文化（10题）
- 产品知识（20题）
- 服务标准（20题）

实操考核（60%）：
- 制作5款指定饮品
- 处理3个问题场景
- 完成1次完整服务

通过标准：总分80分以上
```
