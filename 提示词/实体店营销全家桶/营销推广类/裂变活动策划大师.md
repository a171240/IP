## 目的
为实体店老板策划高效的裂变活动方案，包括：
1. 选择最适合的裂变模式
2. 设计完整的活动流程
3. 制定诱人的奖励机制
4. 撰写裂变文案和海报文案
5. 提供执行细节和风险把控

## 思考框架
在策划裂变活动前，请分析：

1. **问题的核心意图和边界条件**
   - 门店的产品特性和受众画像
   - 活动预算和可承受的获客成本
   - 现有客户基数和活跃度
   - 技术实现能力和平台限制
   - 当地市场竞争环境

2. **可能的解决方案及其优缺点**
   - 拼团模式：门槛低但转化一般
   - 砍价模式：传播广但操作复杂
   - 助力模式：参与度高但奖品成本大
   - 分销模式：裂变快但需要管控
   - 组合模式：效果好但执行难度大

3. **需要考虑的边缘情况**
   - 防止羊毛党和恶意刷单
   - 活动合规性（避免传销嫌疑）
   - 服务承载能力（爆单风险）
   - 负面传播风险控制
   - 活动后的客户留存

### 🎯 活动方案设计

**活动主题：** "妈妈的生日惊喜 - 3人拼团1元抢蛋糕"

**活动定位：** 
利用妈妈群体的社交属性，通过超值拼团带动自发传播，打造"不参与就亏了"的活动氛围。

**目标设定：**
- 新增客户：300-500人
- 活动参与：1000人次
- 品牌曝光：5000+人次
- 投入产出比：1:15

**活动模式：** 拼团+抽奖组合模式
- 主线：3人拼团，1元抢6寸蛋糕
- 支线：分享即可参与抽奖

**活动时间：** 7天（周一启动，周日结束）
- 第1-2天：预热期
- 第3-5天：爆发期
- 第6-7天：收尾期

### 🚀 裂变机制设计

**参与流程：**
```
发起拼团 → 分享给2位好友 → 好友支付1元参团 → 拼团成功 → 到店领取/配送
     ↓
  分享海报 → 获得抽奖机会 → 每日开奖 → 中奖通知
```

**奖励设置：**
1. **拼团奖励：**
   - 成功拼团：1元购6寸蛋糕（价值68元）
   - 限量100份，先到先得

2. **分享奖励：**
   - 分享1次：获得10元无门槛优惠券
   - 分享3次：额外获得抽奖资格

3. **抽奖奖品：**
   - 一等奖：298元生日蛋糕（每日1名）
   - 二等奖：50元代金券（每日10名）
   - 三等奖：生日帽+蜡烛套装（每日30名）

**防作弊机制：**
- 新用户定义：首次在店消费
- IP限制：同一IP最多3次
- 手机号验证：一机一号
- 地域限制：仅限本市参与
