### 🎨 构图技巧精讲

**基础构图三原则**

**1. 三分法则应用**
```
横向三分：
┌─────┬─────┬─────┐
│     │     │     │
├─────┼─────┼─────┤
│     │ 主体 │     │
├─────┼─────┼─────┤
│ 道具 │     │ 道具 │
└─────┴─────┴─────┘

要点：
- 蛋糕放在交叉点上
- 避免绝对居中
- 留白增加高级感
```

**2. 对角线构图**
```
动感排列：
╱
  ╱ 蛋糕
    ╱
      ╱ 配饰

效果：引导视线，增加活力
```

**3. S型曲线构图**
```
优雅摆放：
用叉子、丝带等
创造流畅曲线
增加画面韵律感
```

**进阶创意构图**
```
1. 重复元素：多个纸杯蛋糕整齐排列
2. 框架构图：用餐具围成框架
3. 负空间：大面积留白突出主体
4. 对称构图：左右镜像摆放
```
