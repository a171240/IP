## 输出示例
【美容院营销自动化流程方案】

一、客户旅程地图
```
潜客阶段 → 新客转化 → 活跃期 → 沉睡预警 → 流失召回
   ↓           ↓         ↓         ↓           ↓
 获取线索    首次到店   复购培养   唤醒营销    挽回策略
```

二、自动化场景设计

1. **新客欢迎系列**
- 触发：首次消费后
- T+0：欢迎短信+新客礼包
- T+3：护理建议+专属优惠
- T+7：回访关怀+预约提醒

2. **生日惊喜营销**
- 触发：生日前7天
- T-7：生日祝福预告
- T-0：专属生日礼+双倍积分
- T+3：礼品领取提醒

3. **消费后关怀**
- 触发：每次消费后
- T+1：满意度调查
- T+7：护理效果回访
- T+14：下次护理提醒

4. **会员升级通知**
- 触发：达到升级条件
- 即时：升级祝贺+新权益介绍
- T+1：专属特权体验券

5. **沉睡客户唤醒**
- 触发：30天未到店
- T+30：专属问候+优惠券
- T+37：限时特惠最后提醒
- T+45：升级为流失预警

三、触发规则配置
```python
# 新客欢迎系列
if 客户标签 == "新客" and 首次消费完成:
    发送欢迎短信()
    设置定时任务(T+3, 发送护理建议)
    设置定时任务(T+7, 发送预约提醒)

# 智能预约提醒
if 距离上次消费天数 >= 平均消费周期 * 0.8:
    发送预约提醒()
    赠送限时优惠券()
```

五、效果监测指标
- 自动化覆盖率：触达客户占比
- 消息打开率：各类消息的阅读情况
- 转化率：消息到到店的转化
- ROI：自动化营销的投入产出比
- 退订率：监控用户反感度

优化建议：
- 打开率<30%：优化文案和发送时间
- 转化率<5%：调整优惠力度
- 退订率>5%：降低发送频率
