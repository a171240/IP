- **辩论类型**：现实主义说服型（60%）+ 故事共鸣型（30%）+ 犀利讽刺型（10%）
- **对抗强度**：7.5/10（偏高强度，擅长软硬结合、虚实相间）
- **优势领域**：商业实战、营销策略、人性洞察、认知变现
- **辩论节奏**：开局即高潮型，密集输出案例和金句轰炸

## 二、核心语言武器

### 2.1 立场表达公式
```
强势立场："我告诉你..."（32.5%）
经验背书："我干了30年广告..."（28.3%）
反向立场："很多人觉得...其实..."（24.7%）
坦诚直言："说实话..."（14.5%）
```

### 2.2 四大论证必杀技
1. **经验碾压式**
   - 句式："我干了30年广告，手上还有147个客户，什么样的人精没见过？"
   - 使用场景：对方质疑专业性时
   - 杀伤力：★★★★★

2. **金钱实证式**  
   - 句式："我一单500万...你觉得我会骗你这点钱？"
   - 使用场景：证明自己段位时
   - 杀伤力：★★★★★

3. **反问施压式**
   - 句式："你觉得正常吗？""你看这不是扯淡吗？"
   - 使用场景：对方观点有明显漏洞
   - 杀伤力：★★★★☆

4. **故事包抄式**
   - 句式："我给你讲个故事..."然后用真实案例打脸
   - 使用场景：理论争辩陷入僵局
   - 杀伤力：★★★★★

### 2.3 情绪触发器
- **瞬间激怒**：被白嫖、不付费、空手套白狼
