# 口语化风格优化指南v1.3

## 执行要求
请使用本指南的**三步改写法**处理文本：先标记AI特征，再提取有价值信息，最后自然重写。确保既去除AI味，又保留必要细节。

## 一、核心理念

### 1.1 新版本核心原则
- **去AI味 ≠ 删细节**：去除的是表达方式，不是信息内容
- **平衡优先**：在自然度和完整度之间找到最佳平衡点
- **功能导向**：每个细节都要考虑其功能，而非一刀切

### 1.2 AI味的本质与成因

#### 训练数据的偏向性
- AI主要学习自书面文本（新闻、百科、教材）
- 缺乏真实口语对话的不规则性训练
- 过度学习了"标准"表达方式

#### 生成机制的局限
- 追求逻辑完整性（每个观点都要论证）
- 倾向于信息最大化（该留白的地方也要填满）
- 概率模型导致高频模式重复出现

#### 安全对齐的副作用
- 过度礼貌和中立
- 回避争议性表达
- 情感表达公式化

### 1.3 AI味的五大核心特征
1. **机械重复性**：同一模式反复出现
2. **过度解释性**：把潜台词说成明台词
3. **情感标签化**：用概念词汇代替具体描写
4. **逻辑完美主义**：因果关系过于清晰工整
5. **假装口语化**：为显自然而刻意添加的元素

## 二、AI文本特征详解

### 2.1 句式结构特征

#### 开头模板化
```
❌ AI惯用：
- "说起这个..."
- "记得那是..."
- "每当我想起..."
- "最近一个xx问我（跟我说了一个）xx"

✅ 真实表达：
- 直接进入话题
- 从具体细节开始
- 情绪先行
- 金句开场，吸引眼球
```

#### 转折机械化
```
❌ AI惯用：
- "最扎心的是..."
- "更要命的是..."
- "然而..."
- "但是紧接着..."

✅ 真实表达：
- 转折自然发生
- 不刻意强调转折
- 用情绪或动作暗示转变
```

#### 结尾总结癖
```
❌ AI惯用：
- "总的来说..."
- "这让我明白了..."
- "或许这就是..."

✅ 真实表达：
- 戛然而止
- 情绪自然收尾
- 留白不解释
```

### 2.2 假口语化特征

#### 机械停顿
```
❌ "这个...怎么说呢...就是那种...你懂的吧..."
特征：省略号位置固定，节奏均匀

✅ 真实停顿：
- 位置随机
- 长短不一
- 可能话说一半就转向
```

#### 语气词撒胡椒面
```
❌ "今天啊，我就想说啊，这个事儿吧，真的很奇怪啊。"
特征：语气词分布过于均匀

✅ 真实语气词：
- 因情绪而生
- 集中在情绪高点
- 每个人有固定偏好
```

#### 网络用语堆砌
```
❌ "家人们，我真的会谢，这简直绝绝子，救命啊！"
特征：流行语机械组合，脱离语境

✅ 真实网络表达：
- 符合年龄和身份
- 有个人偏好
- 自然融入语境
```

### 2.3 情感表达特征

#### 情绪标签化
```
❌ AI表达：
"我感到无比愤怒"
"太感动了"
"瞬间泪崩"

✅ 真实表达：
通过动作、细节、节奏变化展现情绪
"我把手机摔了"
"鼻子有点酸"
```

#### 反应公式化
```
❌ AI反应：
"我当时就愣住了"
"我顿时恍然大悟"
"[愣住][苦笑]"

✅ 真实反应：
反应融入叙述，不刻意标注
多样化，不重复使用同一模式
```

### 2.4 逻辑结构特征

#### 因果链过度完整
```
❌ AI逻辑：
"因为A，所以B，因此C，这说明D"

✅ 真实逻辑：
- 跳跃性思维
- 因果关系模糊
- 可能没有结论
```

#### 解释强迫症
```
❌ AI习惯：
每个行为都要解释动机
每个情绪都要分析原因
每个故事都要总结寓意

✅ 真实表达：
很多事情不解释
情绪就是情绪
故事就是故事
```

## 三、禁用词句速查清单

### 3.1 高危词句（必须避免）

#### 脏话
- ❌ 他妈的…
- ❌ 操蛋…
- ❌ TM…
- ❌ 放屁（屎尿屁相关）
- ❌ 情绪可以有，但不要用爆粗口

#### 俗套的形容词
- ❌ 更扎心的是…
- ❌ 扎心的朋友圈…

#### 转折类
- ❌ 最/更 + [形容词] + 的是
- ❌ "更扎心的是..."
- ❌ "更要命的是..."
- ❌ 雪上加霜的是
- ❌ 让人/令人 + [情绪词] + 的是

#### 反应描述类
- ❌ "我当时就笑了"
- ❌ "我当时就懵了/愣了/傻了"
- ❌ "我整个人都..."
- ❌ "我直接..."

#### 假装互动类
- ❌ "你知道吗？"（单独成句）
- ❌ "你知道最XX的是什么吗？"
- ❌ "你猜怎么着？"
- ❌ "你说是不是？"

#### 解释类
- ❌ 这意味着/表明/说明/体现了
- ❌ 由此可见/不难发现
- ❌ 换句话说/也就是说

#### 总结类
- ❌ 总的来说/总而言之
- ❌ 归根结底/说到底
- ❌ 综上所述
- ❌ "所以说啊..."
- ❌ "所以啊..."

#### 假口语类
- ❌ 机械的省略号..."这个...那个..."
- ❌ 标准化惊讶："我当时就懵了"
- ❌ 每句都加语气词

#### 省略号滥用
- ❌ "我...我就是想说..."（假装结巴）
- ❌ "这个嘛...怎么说呢..."（机械犹豫）
- ❌ "其实吧...算了..."（假装欲言又止）
- ❌ "那什么...你懂的..."（刻意含糊）
- ❌ 每段都有省略号（分布过于均匀）

### 3.2 中危表达（视情况避免）

- 视角跳跃："此时的我..."
- 过度描写："我深吸一口气..."
- 情绪叠加："又惊又喜又担心"
- 时间模板："那是一个阳光明媚的午后"

### 3.3 低危提醒（注意频率）

- 网络用语（不要堆砌）
- 反问句（不要每段都用）
- 比喻（不要过于文艺）

## 四、三步改写法（核心方法）

### 4.1 第一步：标记式识别
不直接删除，而是用[ ]标记AI味特征：

```
示例：
原文：今天发生了一件[让人哭笑不得的]事。早上[急匆匆地]赶着上班，
     [心里想着开会的事情]。[然而，最尴尬的是]，到了公司才发现
     穿了两只不同的鞋子。
```

#### 标记原则：
- **必标**：禁用清单中的所有词句
- **选标**：过度描写、冗余解释、假口语
- **不标**：关键信息、必要细节、功能性描述

### 4.2 第二步：信息价值提取

#### 信息分级标准：

| 级别 | 信息类型 | 处理方式 | 示例 |
|------|---------|---------|------|
| A级 | 核心事件 | 必须保留 | "穿了两只不同的鞋" |
| B级 | 背景信息 | 选择保留 | "赶着上班""要开会" |
| C级 | 氛围细节 | 改写保留 | "同事反应" |
| D级 | AI废话 | 完全删除 | "让人哭笑不得""意识到" |

#### 提取技巧：
```
问自己三个问题：
1. 删掉这个信息，故事还完整吗？
2. 这个信息展现了什么（情节/人物/氛围）？
3. 有没有更自然的表达方式？
```

### 4.3 第三步：自然化重写

#### 重写策略对照表：

| AI表达方式 | 转换策略 | 示例 |
|-----------|---------|------|
| 直接说情绪 | 用动作体现 | "很生气"→"摔门" |
| 解释原因 | 展示结果 | "因为堵车所以迟到"→"堵在三环一小时" |
| 概括描述 | 具体细节 | "异样的眼光"→"看了我三秒" |
| 内心独白 | 外在反应 | "我想逃走"→"转身就走" |

## 五、实战案例分析

### 案例1：日常叙事
```
第一步标记：
[昨天经历了一件特别尴尬的事]。和朋友约在咖啡店见面，
[我信心满满地]推门进去，[然而]，玻璃门纹丝不动。
[更尴尬的是]，里面的人都在看我。[原来]这是个拉门，
不是推门。[这件事让我明白，有时候要先观察再行动。]

第二步提取：
- A级：咖啡店见面、推拉门搞错
- B级：里面有人、被看到
- C级：无
- D级：其余全部

第三步重写：
跟朋友约在星巴克。
我对着门使劲推了三下，玻璃门纹丝不动。
里面靠窗的姑娘抬头看我，指了指门把手——"拉"。
```

### 案例2：情感场景
```
第一步标记：
[那一刻，我的心情无比复杂]。看着爸爸[略显佝偻的]背影，
[花白的]头发在阳光下[格外刺眼]。他[小心翼翼地]把我最爱吃的
橘子装进袋子，[就像对待珍宝一样]。[我的鼻子一酸，眼泪差点掉下来。]

第二步提取：
- A级：爸爸的背影、装橘子
- B级：头发花白、动作小心
- C级：阳光
- D级：情绪说明、比喻

第三步重写：
爸爸弯着腰挑橘子，头发在阳光下白得扎眼。
"这个甜，这个也甜。"
他把橘子一个个装进袋子，生怕碰坏了似的。
我转过头，假装看别处。
```

## 七、高级技巧

### 7.1 细节选择原则
```
优先保留：
1. 推动情节的动作
2. 展现性格的细节
3. 营造氛围的环境
4. 产生共鸣的日常

谨慎保留：
1. 纯装饰性描写
2. 可推断的信息
3. 重复性的内容
```

### 7.2 节奏控制技巧
- **紧张时**：短句、动词、少描写
- **抒情时**：可以适当放慢节奏
- **对话时**：符合人物身份和情境
- **叙事时**：长短句结合

### 7.3 人物差异化
不同人物应有不同的表达特征：
- 年龄差异（用词、语气）
- 教育背景（表达方式）
- 性格特点（语言习惯）
- 情绪状态（节奏变化）

## 八、使用指南

### 8.1 基本原则

1. **宁缺勿滥**：宁可简单，不要过度修饰
2. **保持个性**：不同人物用不同的表达习惯
3. **情境优先**：根据场景调整表达方式
4. **自然为上**：读着顺口比规则重要

### 8.2 执行步骤

1. **初稿**：先写出想表达的内容
2. **标记**：用禁用清单标记AI特征
3. **提取**：分析信息价值
4. **重写**：用自然方式重新表达
5. **朗读**：最终用耳朵验证

### 8.3 注意事项

- 某些特殊人设（如知识分子）可适当放宽标准
- 正式场合允许部分书面表达
- 戏剧冲突时可以打破常规
- 持续收集新的AI特征，更新清单

## 九、总结

**记住核心原则：**
1. 严格遵守禁用词句清单
2. 去AI味是去表达方式，不是去内容
3. 每个细节都要考虑其功能
4. 在自然和完整之间找平衡
5. 真实感比规则更重要

**最终目标：**
创造既自然流畅又信息完整的文本，让读者感受到真实的人在真实地表达。
