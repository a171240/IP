# 设计系统色板（品牌级配色规范）

## 1) 中性色
- Ink / 主文字：`#1C1B25`（--text）
- Ink Muted / 次级文字：`#4B4F66`（--muted）
- Ink Subtle / 说明文字：`#6D7288`（--subtle）
- Mist / 主背景：`#F8F6FF`（--bg）
- Cloud / 次背景：`#F1F4FF`（--bg-2）
- Blush / 柔和底色：`#FDF6F8`（--bg-3）
- Glass / 玻璃面：`rgba(255,255,255,0.78)`（--glass）
- Glass Strong / 强玻璃面：`rgba(255,255,255,0.92)`（--glass-strong）
- Border / 细线：`rgba(28,27,37,0.08)`（--border）

## 2) 品牌主色
- Violet / 品牌主色：`#7C3AED`（--brand-violet）
- Indigo / 辅助冷色：`#5B5EF7`（--brand-indigo）
- Blue / 信息与科技：`#3B82F6`（--brand-blue）
- Rose / 活力点缀：`#EC4899`（--brand-rose）
- Coral / 轻盈暖色：`#F97316`（--brand-coral）
- Teal / 专业清爽：`#14B8A6`（--brand-teal）
- Gold / 付费与高级：`#F4B76A`（--brand-gold）
- Slate / 中性平衡：`#64748B`（--brand-slate）

## 3) 语义色
- Success：`#10B981`
- Warning：`#F59E0B`
- Error：`#EF4444`
- Info：`#3B82F6`

## 4) 模块主渐变（建议）
- 工作台：`#7C3AED → #3B82F6`
- IP工厂：`#7C3AED → #EC4899`
- 小红书：`#F97316 → #EC4899 → #F4B76A`
- 我的：`#14B8A6 → #7C3AED`
- 诊断：`#10B981 → #3B82F6`
- 订阅支付：`#F4B76A → #7C3AED`

## 5) Atelier 深色主题（IP工厂高定流程）
- 背景：`#050507`
- 金色主调：`#E5D3B3`
- 珍珠白：`#F5F3FF`
- 深灰：`#2A2A35`
- 玻璃面：`rgba(18,18,22,0.68)`
- 描边：`rgba(229,211,179,0.18)`
- 文本：`#F5F5F5`
- 次文本：`#B9B3C2`

## 6) 使用原则
- 文字默认使用 Ink 系列，避免黑色直接压在深色背景上。
- 渐变使用 2~3 色，避免过度饱和；仅在重点组件（Hero、CTA）使用。
- 玻璃面必须搭配细边框与轻微内阴影，保证层次感与可读性。
